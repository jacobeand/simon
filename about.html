    Símon es un juego de memoría en que sigues el demonstración de un secuencia de colores hasta que equivocas. cuanto más larga sea la secuencia que repita, mayor será el resultado.

    el nombre Símon no es un marca comercial, pero el nombre Simon si es un marca comercial propio de Milton-Bradley. Nuestro uso de este partido es solamente para el uso de educacional sin-ganancia. No vamos a usar este code o programa a fuera de este definicion.
